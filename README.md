# 💰 Django Expense Tracker

A full-stack expense tracking web application built to explore backend logic, database management, and dynamic front-end data visualization.

## 🚀 Features
* **User Authentication:** Secure login, logout, and registration system.
* **Private Wallets:** Data isolation ensures users can only see and manage their own financial records.
* **CRUD Operations:** Users can Create, Read, Update, and Delete their expenses.
* **Dynamic Dashboards:** Real-time data visualization using Chart.js to display spending by category.
* **Admin Dashboard:** Full backend database management through Django's built-in superuser panel.

## 🛠️ Tech Stack
* **Backend:** Python, Django
* **Database:** SQLite (Development)
* **Frontend:** HTML5, CSS3
* **JavaScript:** Chart.js (for data visualization)

## 🎓 About This Project
This application was developed as a hands-on project to master Full-Stack Web Development, bridging the gap between Python server-side logic and interactive web interfaces.
